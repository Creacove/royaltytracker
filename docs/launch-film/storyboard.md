# Storyboard

Target runtime: `42s`

## 0.00-4.00s

Headline: `Royalty intelligence, finally clear.`

Shots:
- Macro crop of `Reports & Statements` upload zone
- Drag file into upload area
- Fast cut to report row with `processing` then `completed_with_warnings`

Capture notes:
- Keep the file name readable for less than one second
- Show one believable CMO and one reporting period
- Use a cursor path with one clean diagonal move only

## 4.00-8.00s

Headline: `From statements to signal.`

Shots:
- Dashboard KPI strip punch-in on `Net Revenue`
- Drift across `Revenue Trend (12 Months)`
- Quick rack focus to `Platform Revenue Mix`

Capture notes:
- Net revenue must be the hero number
- Trend line should end higher than it starts
- Platform mix should have one dominant platform and one strong secondary

## 8.00-12.00s

Headline: `Ask across your catalog.`

Shots:
- Enter `AI Insights`
- Hold empty state for half a beat
- Cursor lands in composer

Capture notes:
- Capture one clean pass with no typing
- Capture a second pass with manual typing for the first prompt

## 12.00-17.00s

Headline: `Find the leak.`

Prompt:

`Where is revenue leaking the most?`

Shots:
- Type prompt
- Send
- Cut on answer reveal
- Macro crop answer title and first sentence

Answer focus:
- Risk concentrated in one territory
- Commission drag or weak payout signal
- One action-oriented recommendation

## 17.00-22.00s

Headline: `Then narrow the lens.`

Prompt:

`Which artists should we prioritize this quarter?`

Shots:
- Context rail visible in full frame
- Switch from workspace view into artist scope
- Answer appears
- Click artist snapshot from the rail

Answer focus:
- Name one lead artist
- Mention momentum and monetization upside
- Keep sentence count short enough for a `2s` read

## 22.00-27.00s

Headline: `Drill into artists.`

Shots:
- Artist snapshot KPI strip
- Artist revenue trend
- `What deserves attention` signal cards
- Top tracks table sweep

Capture notes:
- One track must clearly dominate
- Top territory and top platform should be legible
- Signal cards should include one growth opportunity and one concentration risk

## 27.00-32.00s

Headline: `Open the track.`

Prompt:

`Show tracks with highest opportunity.`

Shots:
- Jump back to AI Insights answer or transition directly into track snapshot
- Track snapshot summary card
- Territory mix bar chart
- Platform mix bar chart
- Export button hover

Capture notes:
- Track snapshot should feel like the premium hero surface
- Summary card needs one short line worth spotlighting in edit

## 32.00-35.00s

Headline: `Trust the trail.`

Shots:
- Data Quality Queue issue card
- Resolution panel
- Source evidence grid flash

Capture notes:
- Only show one or two issues
- Message should imply auditability, not broken software

## 35.00-38.00s

Headline: `See every line.`

Shots:
- Transactions table sweep
- Open transaction detail drawer
- Hold provenance block for less than one second

Capture notes:
- Choose a row with clean ISRC, territory, platform, and revenue values
- The drawer should validate the AI story, not introduce noise

## 38.00-42.00s

Headline: `Export the snapshot.`

Shots:
- Trigger snapshot export
- Hold downloaded artifact moment
- End on product lockup

End card:
- Line 1: `RoyaltyTracker`
- Line 2: `Instant decisions from royalty data.`

## Edit Rhythm

- Average shot length: `1.2-2.2s`
- Hero holds: artist snapshot, track snapshot, end card
- Preferred transitions: hard cuts, directional wipes, masked crop pushes, speed ramps
- Avoid: glitch FX, neon AI motifs, floating 3D chrome, fake terminal overlays
